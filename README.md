# wp-plugins

Premium and third-party WordPress plugins downloaded from vendor websites or repositories.

## Plugins

- [Advanced Custom Fields Pro](acf-pro/advanced-custom-fields-pro.zip)
- [WPML](wpml/sitepress-multilingual-cms.4.9.2.zip)
- [WPGraphQL Content Blocks](wp-graphql-content-blocks/wp-graphql-content-blocks.zip)

## Usage

Download the latest `.zip` from the vendor, place it in the appropriate directory, and deploy:

```bash
wp plugin install path/to/plugin.zip --activate
```

Premium plugins (ACF Pro, WPML) require valid licenses for downloads and updates.
